Using the Run Script

If you need help with this section, see:

  • Mac command line
  • Ubuntu command line

Once you have already run the program for the first time, you may use the provided run script to make future runs as simple as double clicking a file on your desktop. This feature is limited to Mac and Linux operating systems.

The run script will only work if you have installed the program directory in your home directory (that is, the path to the program directory must be ~/otf_data). You may save the run script wherever you like, though.

Using the Run Script on Mac

  1. Download the run script.
  2. Make the file executable.
  3. Choose to either just once or always open this type of file with the Terminal application.

Using the Run Script on Ubuntu

  1. Download the run script.
  2. Make the file executable.
  3. Choose to either just once or always open this type of file with the Terminal application.
