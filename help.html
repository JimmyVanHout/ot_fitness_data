Installing the Program

The recommended approach to installing the program is via the command line:
git clone https://github.com/JimmyVanHout/otf_data.git
For help, see:

  • Mac command line
  • Ubuntu command line
  • Windows PowerShell command line

An alternative method to install the program is by downloading the program from GitHub.

Installing the Program Requirements

The recommended approach to installing the program requirements is via the command line:
cd <program_directory_path> pip3 install flask pip3 install bs4
where <program_directory_path> is the path to the program directory. If you installed the program in your home directory, then <program_directory_path> should be ~/otf_data. For help, see:

  • Mac command line
  • Ubuntu command line
  • Windows PowerShell command line

More detailed help and alternative installation methods are available below:

  • Mac
  • Ubuntu
  • Windows

Mac

You can install the requirements either via the command line (recommended) or the provided install script:

Command Line

Run the following on the command line:
cd <program_directory_path> pip3 install flask pip3 install bs4
where <program_directory_path> is the path to the program directory.

Install Script

  1. Download the install script.
  2. Make the file executable.
  3. Choose to either just once or always open this type of file with the Terminal application.

Running the Program

The recommended approach to running the program is via the command line:
cd <program_directory_path> python3 otf_data.py
where <program_directory_path> is the path to the program directory. If you installed the program in your home directory, then <program_directory_path> should be ~/otf_data. For help, see:

  • Mac command line
  • Ubuntu command line
  • Windows PowerShell command line

More detailed help and alternative methods to run the program are available below:

  • Mac
  • Ubuntu
  • Windows

Mac

You can run the program either via the command line (recommended) or the provided helper script:

Command Line

Run the following on the command line:
cd <program_directory_path> python3 otf_data.py
where <program_directory_path> is the path to the program directory.

Helper Script

To use the helper script, you must rename the directory of the downloaded data-extracting program to otf_data and save this directory to your desktop (which should be located at ~/Desktop).

Then proceed with the following steps:

  1. Download the helper script.
  2. Make the file executable.
  3. Choose to either just once or always open this type of file with the Terminal application.
