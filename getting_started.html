Getting Started

About

This site analyzes the data from your OrangeTheory Fitness workout summary emails and displays the results. It runs entirely in your browser, so your data stays on your computer--it is not sent to a server.

To extract the data from the emails, you will run a program that does this, as described in the steps below. You will only have to complete the sequence of steps in the First Time section once. After that, running the program is much simpler (see Next Time), depending on the settings you choose during the configuration.

First Time

In short, the recommended approach described below is to create a new Gmail account to use only for your workout summary emails. Past emails can be forwarded from your usual account to this new account using the data-extracting program, and future emails can be forwarded using a filter. After the program runs, you will upload the newly created data file to this site, which will analyze and display the data for you.

Below are the detailed instructions. If you need help, please visit the Help page of this site.

Note: When downloading the following files, ensure they are all saved to the same directory.

Install the Program and Requirements

  1. Download the latest version of Python.

  2. The data-extracting program is available on GitHub, and, using the command line and ensuring there are no files in the current directory named otf_data, this is the recommended way to install it:
    git clone https://github.com/JimmyVanHout/otf_data.git
    See the section of the Help page on installing the program for help. For your convenience you may also download the program from GitHub here.

    This program extracts the workout data from the emails to a CSV file. For a more detailed overview of how the program works, read the README file (the next several steps are from that document). Important to note:

    1. To avoid potential loss of data, keep the program files within the program directory (this is because the program writes the files config.txt and data.csv and will overwrite any existing files with these names in the same directory).

    2. The program saves the password of the new Gmail account you will create as plain text on your computer. This shouldn't be an issue as long as proper security measures are taken, although if you would like you can disable this feature by running the program with the correct option (see the README; note that this prevents the configuration file from being saved and will therefore require configuration of the program each time it is run).

  3. Install the program requirements. The recommended approach is via the command line:
    cd <program_directory_path> pip3 install flask pip3 install bs4
    where <program_directory_path> is the path to the program directory.

    See the section of the Help page on installing the program requirements for help and alternative ways to install the requirements.

Set up the Gmail Configuration

  1. Create a new Gmail account. This account will be used to receive the workout summary emails. While creating a new account is not absolutely necessary, for security considerations and ease of use this is strongly recommended.

  2. On the Gmail account that typically receives the workout summary emails, set up a filter to forward all future mail from otbeatreport@orangetheoryfitness.com to the new Gmail account you just created. Since a Gmail filter cannot forward past mail, the data-extracting program has the ability to do this.

  3. In order for the data-extracting program to be able to read from the new Gmail account, you must disable two-factor authentication and enable "less secure app" access on the new account in the security settings for that account.

Run and Configure the Program

  1. As when installing the program requirements, the recommended approach to run the program is again via the command line:
    cd <program_directory_path> python3 otf_data.py
    where <program_directory_path> is the path to the program directory.

    See the section of the Help page on running the program for help and alternative ways to run the program.

    Once you run the program for the first time, a configuration menu will appear:

    1. If you want to forward your past workout summary emails from your default account to your new account, select the option to do so and fill out the information.

    2. To make the program run faster in the future, select the option to only search for new workout summary emails.

    After filling out the information, submit the form. The program will run and generate the data file. If any records contained errors, they will be displayed and not included in the file.

  2. Once the data file has been generated, upload it to the Data page of this site. The data will be analyzed and presented to you.

Next Time

If you choose to leave two-factor authentication off and keep "less secure app" access turned on for your new Gmail account, whenever you want to see your newest data on this site in the future all you will need to do is:

  1. Run the program. The recommended approach is via the command line:
    cd <program_directory_path> python3 otf_data.py
    where <program_directory_path> is the path to the program directory.

    See the section of the Help page on running the program for help and alternative ways to run the program.

  2. Upload the data file to the Data page.

You may you choose to turn on two-factor authentication and turn off "less secure app" access after you run the data-extracting program, just note that you will need to change these settings whenever you want to run the program in the future.
