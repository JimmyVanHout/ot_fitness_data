Getting Started

About

This site analyzes the data from your OrangeTheory Fitness workout summary emails and displays the results. It runs entirely in your browser, so your data stays on your computer--it is not sent to a server.

To extract the data from the emails, you will run a program that does this, as described in the steps below. You will only have to complete the sequence of steps in the First Time section once. After that, running the program is much simpler (see Next Time), depending on the settings you choose during the configuration.

First Time

In short, the recommended approach described below is to create a new Gmail account to use only for your workout summary emails. Past emails can be forwarded from your usual account to this new account using the data-extracting program, and future emails can be forwarded using a filter. After the program runs, you will upload the newly created data file to this site, which will analyze and display the data for you.

The program extracts the workout data from the emails to a CSV file. For a more detailed overview of how the program works, read the README file (the next several steps are from that document). Important to note:

  1. To avoid potential loss of data, keep the program files within the program directory (this is because the program writes the files config.txt and data.csv and will overwrite any existing files with these names in the same directory).

  2. The program saves the password of the new Gmail account you will create as plain text on your computer. This shouldn't be an issue as long as proper security measures are taken, although if you would like you can disable this feature by running the program with the correct option (see the README; note that this prevents the configuration file from being saved and will therefore require configuration of the program each time it is run).

Follow the steps below to get started:

Set up the Gmail Configuration

  1. Create a new Gmail account. This account will be used to receive the workout summary emails. While creating a new account is not absolutely necessary, for security considerations and ease of use this is strongly recommended.

  2. On the Gmail account that typically receives the workout summary emails, set up a filter to forward all future mail from otbeatreport@orangetheoryfitness.com to the new Gmail account you just created. Since a Gmail filter cannot forward past mail, the data-extracting program has the ability to do this.

  3. In order for the data-extracting program to be able to read from the new Gmail account, you must disable two-factor authentication and enable "less secure app" access on the new account in the security settings for that account.

Install the Program and Dependencies

If you need any help with the instructions below, see:

  • Mac command line
  • Ubuntu command line
  • Windows PowerShell command line

  1. Download the latest version of Python.

  2. Run the following on the command line:
    git clone https://github.com/JimmyVanHout/otf_data.git pip3 install flask pip3 install bs4

Run and Configure the Program

If you need any help with the instructions below, see:

  • Mac command line
  • Ubuntu command line
  • Windows PowerShell command line

  1. Run the following on the command line:
    cd <program_directory_path> python3 otf_data.py
    where <program_directory_path> is the path to the program directory. For example, if you saved cloned the program into your home directory, this might be ~/otf_data.

  2. Once you run the program for the first time, a configuration menu will appear. The first checkbox asks whether you want to forward past emails to your new account:

    Choosing to Forward

    If you choose to forward past emails, fill out the first section, titled "One-Time Forwarding of All Past OT Beat Report Emails from This Account", in the following way:

    • Enter the email address of your usual Gmail account (the forwarding account).

    • Enter the password of your usual Gmail account (the forwarding account).

    • Confirm the password of your usual Gmail account (the forwarding account).

    • Enter the email address of your new Gmail account (the receiving account).

    Now fill out the next section, titled "Then Receive Them Here and Get OT Beat Report Data from This Account", in the following way:

    • Enter the email address of your new Gmail account.

    • Enter the password of your new Gmail account.

    • Confirm the password of your new Gmail account.

    • Under "Comma-separated originating email addresses", enter otbeatreport@orangetheoryfitness.com, (including the comma) followed by the address of your usual Gmail account.

    Not Choosing to Forward

    If you choose not to forward, fill out the section titled "Get OT Beat Report Data from This Account", in the following way:

    • Enter the email address of your new Gmail account.

    • Enter the password of your new Gmail account.

    • Confirm the password of your new Gmail account.

    • Under "Comma-separated originating email addresses", enter otbeatreport@orangetheoryfitness.com, followed by the address of your usual Gmail account.

    Regardless of whether you choose to forward past emails or not, select the option to only search for new workout summary emails to make the program run faster in the future.

    After filling out the information, submit the form. The program will run and generate the data file, named data.csv, in the program directory, otf_data.

    Note:

    • If any records contained errors, they will be displayed and not included in the file.

    • If you accidentally submitted incorrect information, simply delete the config.txt file in the program directory (otf_data) and then run the program again.

  3. Once the data file has been generated, upload it to the Data page of this site. The data will be analyzed and presented to you.

Next Time

If you choose to leave two-factor authentication off and keep "less secure app" access turned on for your new Gmail account, in the future whenever you want to see your newest data on this site all you will need to do is:

  1. Run the program on the command line (you will not have to complete the configuration again):
    cd <program_directory_path> python3 otf_data.py
    where <program_directory_path> is the path to the program directory. For example, if you installed the program in your home directory, this might be ~/otf_data.

  2. Upload the data file to the Data page.

You may you choose to turn on two-factor authentication and turn off "less secure app" access after you run the data-extracting program, just note that you will need to change these settings whenever you want to run the program in the future.
